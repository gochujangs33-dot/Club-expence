# 클럽 비용 정산기 — 계산식 명세서

> 본 문서는 클럽 비용 정산기 웹 앱(`app.js`)에 실제로 적용된 계산 로직을 정리한 명세서입니다.

---

## 1. 비용 항목 분류

| 카테고리 | 코드 | 설명 |
|---------|------|------|
| 행사비 | `EVENT` | 식대, 경기 진행비 등 행사 운영비 |
| 시설 및 장비 이용료 | `FACILITY` | 시설 대여, 장비 사용료 |
| 상품 | `PRIZE` | 시상품, 경품 등 |

---

## 2. 비용 집계 계산식

```
총 소요 비용 = Σ(모든 비용 항목 금액)
행사비 합계  = Σ(행사비 항목 금액)
시설 이용료  = Σ(시설 장비 항목 금액)
상품비 합계  = Σ(상품 항목 금액)

인당 행사비  = 행사비 합계 ÷ 참석자 수
```

---

## 3. 인당 자부담 계산식 (구간별 누진 방식)

> **기준값 (설정 화면에서 변경 가능)**
> - 가. 지원 한도: **30,000원 이하**
> - 나. 구간 상한: **60,000원 이하**, 자부담율 **20%**
> - 다. 구간 상한: **120,000원 이하**, 자부담율 **40%**
> - 라. 초과 시 공제액: **85,000원**

구간 판정 기준은 **인당 행사비**입니다.

| 구간 | 조건 | 인당 자부담 계산식 |
|------|------|--------------------|
| 가 | 인당 행사비 ≤ 30,000원 | **0원** (자부담 없음) |
| 나 | 30,000원 < 인당 행사비 ≤ 60,000원 | `인당 행사비 × 20%` |
| 다 | 60,000원 < 인당 행사비 ≤ 120,000원 | `60,000 × 20% + (인당 행사비 - 60,000) × 40%` |
| 라 | 인당 행사비 > 120,000원 | `인당 행사비 - 85,000원` |

#### 구간 다 계산 예시

```
인당 행사비 = 80,000원 일 때:
  part1 = 60,000 × 20% = 12,000원
  part2 = (80,000 - 60,000) × 40% = 8,000원
  인당 자부담 = 12,000 + 8,000 = 20,000원
```

---

## 4. 전체 자부담 및 정산 결과

```
인당 자부담 = (구간 계산식 결과)
총 자부담   = round(인당 자부담 × 참석자 수)
자부담 비율 = 총 자부담 ÷ 총 소요 비용 × 100 (%)
최종 지원금 = 총 소요 비용 - 총 자부담
```

---

## 5. 총 자부담 금액 수동 수정 시 차액 계산

사용자가 자동 계산된 총 자부담 금액을 직접 수정한 뒤 **확인** 버튼을 누르면 팝업창으로 아래 계산식과 결과를 표시합니다.

```
차액 = 새로 입력한 총 자부담 금액 - 수정 전 총 자부담 금액
```

팝업 출력 형식:

```
70,000원 - 63,000원 = +7,000원
```

수동 수정 시 연동 재계산 항목:

| 항목 | 재계산 식 |
|------|-----------|
| 인당 자부담 비용 | `수정 후 총 자부담 ÷ 참석자 수` |
| 자부담 비율 | `수정 후 총 자부담 ÷ 총 소요 비용 × 100%` |

---

## 6. 경고 조건

| 조건 | 경고 메시지 |
|------|------------|
| 상품비 > 0 이고 참석자 수 < 20명 | 정회원 20명 이상 참석 시에만 상품비 사용 가능 |
| (이번 상품비 + 기존 누적 상품비) > 500,000원 | 상품비 연간 한도 500,000원 초과 |
| 시설 및 장비 이용료 > 1,000,000원 | 별도 협의 필요 |

---

## 7. 누적 참석 횟수 관리

현재 참석자 명부와 누적 사원 명부가 연동되어 올해 누적 참석 횟수를 자동으로 가감합니다.

```
참석자 등록 시: directory[이름].count = count + 1  (해당 행사에 처음 등록하는 경우에만)
참석자 삭제 시: directory[이름].count = max(0, count - 1)
```

누적 사원 명부 탭에서 `(올해 누적: N회)` 배지로 실시간 확인 가능합니다.
