# 사용자 권한 정리 (v1.6.62 기준)

## 1. 일반 회원 (PIN 로그인, 000000/002531 제외)

- 비용 항목 추가/수정/삭제 (법인카드/개인카드/분할)
- 참석자 추가, 전사원 명부에서 검색하여 추가
- 전사원 명부에 **신규 등록**(이름/EMP ID 추가) — 단, 삭제는 불가
- 클럽 이름 직접 등록(미가입 시 신규 클럽명 등록 가능, v1.6.8)
- ⚙️ 설정에서 "기존에 사용한 누적 금액(이전 사용 금액)" 입력
- 정산 확정 → 본인 [정산 이력] 탭에 기록 저장 (엑셀 자동 생성)
- [정산 이력] 탭에서 본인 이력 **조회만 가능, 삭제 불가** (삭제는 관리자에게 요청)
- 메일 앱으로 정산 보고서 전송, 요청사항(피드백) 등록
- 본인 클럽의 예산 소진율 등 차트 조회

## 2. 관리자 (PIN 000000)

일반 회원의 모든 기능 +

- **정산 구간 및 비율 설정** (가/나/다/라 구간, 자부담 비율, 공제액) — 전체 클럽 공통 적용, 엑셀 내보내기에도 자동 반영 (v1.6.60)
- 클럽 관리: 클럽 추가/수정, 클럽별 예산 설정·"추가 배정"
- 클럽별 정산이력(전체 클럽의 globalHistory) 조회
- **정산 기록 삭제** — globalHistory + 작성자 개인 이력에서 함께 제거, 참석자 누적 참석 횟수 차감 (관리자 전용)
- 회원 관리: 회원 정보 수정, 회원 삭제
- 전사원 명부 삭제(× 버튼)
- 요청사항(피드백) 목록 확인
- 관리자 대시보드 차트(월별 추이, 클럽별 예산 소진율, 카테고리 비중, 자부담 추이) 조회

## 3. 개발자 (PIN 002531 / 김종필)

- 로그인 시 **관리자 모드(000000)와 동일하게 전환**됨 (v1.6.63) — 정산 구간/비율 설정, 클럽·예산 관리, 전체 정산이력 조회·삭제, 회원/명부 관리, 대시보드 차트 등 관리자 권한 전부 사용 가능
- 명부 삭제 권한도 동일 부여 (`deleteFromDirectory()` 등)

## 비고

- "정산 이력" 동기화: 회원이 정산 확정 시 본인 이력(`settlements/{pin}/settlementHistory`)과 전체 이력(`globalHistory`)에 동시 저장됨.
  과거(v1.6.55 이전)에 생성되어 누락된 항목은 로그인 시 1회성으로 자동 보충 동기화됨 (v1.6.62).
